# Inuyasha Toggle Switch

<b>Made with:</b><br/>
HTML, CSS & JS
<br/><br/>
<b>What is this?</b><br/>
A cool Inuyasha toggle :)
<br/><br/>
<b>Description:</b><br/>
It's a toggle switch, that shows Inuyasha with black when it's off and Inuyasha with white hair when it's on.
